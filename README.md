# Ecommerce Store Assignment

## Overview

This repository contains a simple **ecommerce application** built as part of an assignment.
The goal of the project is to design and implement core ecommerce functionality using
backend APIs, optional frontend UI, and supporting tools such as tests and Postman collections.

The project is structured into **server**, **client**, and **postman** folders to clearly
separate responsibilities.

---

## Repository Structure

```txt
ecommerce-store/
├── server/     # Backend APIs (Node.js + Express)
├── client/     # Frontend UI (React + Vite)
├── postman/    # Postman collection for API testing
```


## Implemented

- Add items to cart

- Checkout functionality

- Coupon generation on every Nth order

- Coupon validation during checkout

- Admin APIs (coupon generation and summary)
